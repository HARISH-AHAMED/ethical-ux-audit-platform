# Ethical UX Audit Platform

## Team Maverick
Hackathon Project for **Hacksagon 2026**

## Overview
Ethical UX Audit Platform is a web-based tool designed to detect **dark patterns** and unethical UX practices used in websites.

It helps users, designers, and organizations identify manipulative interface elements and improve digital trust.

## Problem Statement
Many websites use deceptive UI/UX techniques such as:
- Forced sign-ups
- Hidden cancellation flows
- Misleading CTA buttons
- Fake urgency / countdown pressure
- Pre-checked boxes
- Nagging popups

These patterns reduce user trust and negatively impact ethical web design.

## Solution
Our platform allows users to submit a website URL and receive an **automated ethical UX audit report**.

The report includes:
- Ethical UX Score
- Detected dark patterns
- Severity levels
- Suggestions for improvement

## Key Features
- Website URL-based UX audit
- Detection of common dark patterns
- Ethical UX Score generation
- Dashboard with audit insights
- Future support for audit history and trend tracking

## Proposed Workflow
1. User enters a website URL
2. Frontend sends URL to backend API
3. Backend scans and analyzes the website
4. Detection logic identifies dark patterns
5. Results are generated and displayed in dashboard

## Tech Stack
- **Frontend:** React + Vite
- **Styling:** Tailwind CSS / CSS
- **Backend:** Node.js + Express
- **Database:** MongoDB
- **Version Control:** GitHub

## Future Scope
- Browser extension for live UX auditing
- AI-powered UI screenshot analysis
- Mobile app UX audit support
- Ethical UX benchmarking across websites
